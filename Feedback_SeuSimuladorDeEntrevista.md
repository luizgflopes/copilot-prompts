📅 Plano de Estudo (5h/semana)
Curto Prazo (1–2 meses)

    Objetivo: Dominar fundamentos do Power BI e criar relatórios simples.

    Atividades:

        YouTube – Curso Power BI do Básico ao Profissional (playlist gratuita com 60 aulas)

        Dedicar 2h/semana para assistir aulas e replicar exemplos.

        1h/semana para praticar com dados reais (ex.: exportar bugs do N2 ou cards do board).

        2h/semana para exercícios práticos: criar relatórios básicos (tabelas, gráficos simples).

Médio Prazo (3–4 meses)

    Objetivo: Construir dashboards interativos e aplicar DAX.

    Atividades:

        Udemy – Power BI Completo (Básico ao Avançado), curso estruturado com dashboards dinâmicos.

        Dedicar 3h/semana para aulas e exercícios guiados.

        2h/semana para projetos próprios: montar dashboards de fluxo de cards (Dev, QA, HML) e tempo de ciclo.

        Aprender DAX para criar métricas personalizadas (ex.: tempo médio de resolução de bugs).

Longo Prazo (5–6 meses)

    Objetivo: Integrar BI às decisões de produto e comunicar insights.

    Atividades:

        Microsoft Learn – Treinamento oficial Power BI (roteiros de aprendizagem gratuitos).

        Dedicar 2h/semana para aprofundar em modelagem de dados e boas práticas.

        3h/semana para projetos aplicados: criar dashboards que suportem decisões de backlog (ex.: priorização baseada em volume de bugs ou tempo de entrega).

        Treinar apresentações: explicar insights de forma clara para stakeholders.
